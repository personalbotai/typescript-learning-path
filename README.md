# TypeScript Learning Path

Interactive tutorial untuk belajar TypeScript dari dasar hingga advanced.

## Features

- **10 Modules** lengkap dengan lessons, code examples, dan quiz
- **Monaco Editor** dengan TypeScript syntax highlighting
- **Live Code Execution** via WebContainer API (compile & run TypeScript di browser)
- **Progress Tracking** dengan localStorage
- **Quiz System** dengan scoring
- **Responsive Design** (mobile-first)
- **GitHub Pages Ready**

## Tech Stack

- HTML5, CSS3 (Vanilla, mobile-first)
- TypeScript (tutorial content)
- Monaco Editor (VS Code editor in browser)
- WebContainer API (StackBlitz) untuk code execution
- No build step required - static files only

## Local Development

1. Clone repository
2. Serve dengan static server:
   ```bash
   npx serve .
   ```
3. Buka http://localhost:3000

## Deployment

Repository dikonfigurasi untuk GitHub Pages:

1. Push ke GitHub
2. Settings → Pages → Source: `gh-pages` branch
3. Akan auto-deploy via GitHub Actions (atau manual dengan `gh` CLI)
