Conditional Types

Conditional types memungkinkan kita memilih tipe berdasarkan kondisi. Mirip seperti operator ternary di JavaScript, tapi bekerja di level tipe. Ini sangat powerful untuk membuat tipe yang adaptable berdasarkan input tipe lainnya.

1 Sintaks Dasar

Conditional types menggunakan syntax: T extends U ? X : Y. Jika tipe T dapat ditetapkan ke U, maka hasilnya adalah X, jika tidak maka Y.

type IsString<T> = T extends string ? true : false; type A = IsString<string>; // true type B = IsString<number>; // false type C = IsString<string | number>; // false (karena union tidak extends string)

2 Conditional Types dengan Union (Distributive)

Ketika conditional types diterapkan pada union types, mereka secara otomatis terdistribusi ke setiap anggota union.

type ToArray<T> = T extends any ? T[] : never; // Distribusi: string | number ‚Üí string[] | number[] type Arr = ToArray<string | number>; // string[] | number[]

Ini membantu membuat tipe seperti Flatten<T> atau ReturnType<T> yang bekerja dengan union.

3 Infer (Mengekstrak Tipe)

Kata kunci infer memungkinkan kita mengambil (extract) tipe dari suatu tipe lain di dalam conditional type. Berguna untuk membuat utility types seperti ReturnType, Parameters, dll.

// Extract return type dari function type ReturnType<T> = T extends (...args: any[]) => infer R ? R : never; function greet(name: string): string { return "Hello " + name; } type GreetReturn = ReturnType<typeof greet>; // string // Extract first parameter type FirstParam<T> = T extends (arg: infer P, ...rest: any[]) => any ? P : never; type GreetFirst = FirstParam<typeof greet>; // string

4 Conditional Types Bersarang

Kita bisa menumpuk conditional types untuk membuat logika yang lebih kompleks.

type Flatten<T> = T extends any[] ? T[number] : T extends object ? T extends Function ? T : T extends null | undefined ? T : T : T; // Contoh penggunaan type T1 = Flatten<string[]>; // string type T2 = Flatten<number>; // number type T3 = Flatten<string | number>; // string | number

5 Contoh Praktis: Parameter & ReturnType

TypeScript sudah menyediakan beberapa conditional types bawaan. Mari kita lihat implementasinya:

// Parameters (mengambil tipe parameter dari function) type Parameters<T> = T extends (...args: infer P) => any ? P : never; // ConstructorParameters (untuk class constructor) type ConstructorParameters<T extends new (...args: any[]) => any> = T extends new (...args: infer P) => any ? P : never; // Contoh: function foo(x: number, y: string) { } type FooParams = Parameters<typeof foo>; // [number, string]
