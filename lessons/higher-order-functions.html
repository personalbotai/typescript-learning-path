Modul 3: Functions & Functional Programming

Kemajuan Belajar 0%

Higher-Order Functions (HOF)

Higher-Order Function adalah function yang menerima function sebagai argument atau mengembalikan function. Ini adalah konsep fundamental dalam functional programming.

Konsep: Di TypeScript, HOF memiliki type signatures yang harus didefinisikan dengan jelas. Callback function adalah function yang dilewatkan sebagai argument ke HOF.

1. Function sebagai Argument (Callback)

Function bisa dilewatkan sebagai parameter ke function lain.

// Higher-order function yang menerima callback
function processNumbers(
    numbers: number[],
    callback: (n: number) => number
): number[] {
    let result: number[] = [];
    for (let n of numbers) {
        result.push(callback(n));
    }
    return result;
}

// Callback functions
let doubled = (x: number) => x * 2;
let squared = (x: number) => x * x;

// Gunakan HOF
let arr = [1, 2, 3];
console.log(processNumbers(arr, doubled)); // [2,4,6]
console.log(processNumbers(arr, squared)); // [1,4,9]

// Type signature dari processNumbers:
// (numbers: number[], callback: (n: number) => number) => number[]

2. Function yang Mengembalikan Function

HOF bisa mengembalikan function baru (closure).

// Function factory
function createMultiplier(factor: number): (x: number) => number {
    return (x: number) => x * factor;
}

let double = createMultiplier(2);
let triple = createMultiplier(3);

console.log(double(5));   // 10
console.log(triple(5));   // 15

// Closure: returned function "menangkap" factor dari outer scope
function createGreeting(greeting: string): (name: string) => string {
    return (name: string) => `${greeting}, ${name}!`;
}

let hello = createGreeting("Hello");
let selamat = createGreeting("Selamat pagi");

console.log(hello("Alice"));       // "Hello, Alice!"
console.log(selamat("Budi"));      // "Selamat pagi, Budi!"

Closure

Function yang dikembalikan memiliki akses ke variabel-variabel di scope outer-nya, bahkan setelah outer function selesai dieksekusi. Ini disebut closure.

3. Contoh HOF Populer (Built-in)

JavaScript/TypeScript memiliki banyak built-in HOF di Array prototype.

// map, filter, reduce adalah HOF!
[1,2,3].map(x => x * 2);      // map menerima callback
[1,2,3].filter(x => x > 1);   // filter menerima predicate callback
[1,2,3].reduce((a,b) => a+b, 0); // reduce menerika accumulator callback

// Set, Map juga punya forEach (HOF)
let set = new Set([1,2,3]);
set.forEach(x => console.log(x));

// Function composition dengan HOF
function compose(f: (x: U) => V, g: (x: T) => U): (x: T) => V {
    return (x: T) => f(g(x));
}

let add1 = (x: number) => x + 1;
let double = (x: number) => x * 2;
let add1ThenDouble = compose(double, add1);
console.log(add1ThenDouble(5)); // (5+1)*2 = 12

4. Type Signature untuk HOF

Sangat penting untuk mendefinisikan type callback dengan jelas.

// HOF dengan callback memiliki 2 atau lebih parameters
function filter(
    arr: T[],
    predicate: (value: T, index: number, array: T[]) => boolean
): T[] {
    let result: T[] = [];
    for (let i = 0; i < arr.length; i++) {
        if (predicate(arr[i], i, arr)) {
            result.push(arr[i]);
        }
    }
    return result;
}

// Predicate callback menerima value, index, dan array
let numbers = [1,2,3,4,5];
let evens = filter(numbers, (value, index, arr) => {
    console.log(index, value, arr.length);
    return value % 2 === 0;
});

5. Currying (Preview)

Currying adalah teknik di mana function yang menerima multiple arguments diubah menjadi series of functions yang masing-masing menerima satu argument.

// Regular function
function add(a: number, b: number): number {
    return a + b;
}

// Curried version
function addCurried(a: number): (b: number) => number {
    return (b: number) => a + b;
}

let add5 = addCurried(5); // partial application
console.log(add5(3));     // 8

// Generic curry helper
function curry(fn: (a: T, b: U) => V): (a: T) => (b: U) => V {
    return (a: T) => (b: U) => fn(a, b);
}
let curriedAdd = curry(add);
let add10 = curriedAdd(10);
console.log(add10(5)); // 15

Currying akan dibahas lebih detail di lesson berikutnya.

Latihan: Build Custom Pipeline

// Buat HOF untuk compose multiple functions
function pipe(...fns: Array<(arg: T) => T>): (arg: T) => T {
    return (x: T) => fns.reduce((acc, fn) => fn(acc), x);
}

// Contoh usage
let toUpperCase = (s: string) => s.toUpperCase();
let exclaim = (s: string) => s + "!";
let shout = pipe(toUpperCase, exclaim);

console.log(shout("hello")); // "HELLO!"

// Type-safe pipe untuk berbagai tipe
function pipeGeneric(
    f1: (a: A) => B,
    f2: (b: B) => C
): (a: A) => C {
    return (a: A) => f2(f1(a));
}

Kesalahan Umum

Lupa type callback

function badHOF(arr, cb) { // any types!
    return arr.map(cb);
}
// Selalu specify types!

Callback hell

// Terlalu nested callback
doA(() => {
    doB(() => {
        doC(() => { ... });
    });
});
// Gunakan promise/async-await atau compose

Rangkuman

  • Higher-Order Function: function yang menerima atau mengembalikan function
  • Callback: function yang dilewatkan sebagai argument ke HOF
  • Built-in HOF: map, filter, reduce, forEach
  • Function yang mengembalikan function menghasilkan closure
  • Type signature untuk callback harus didefinisikan dengan jelas (params dan return type)
  • Currying adalah teknik HOF untuk partial application (akan dibahas lebih lanjut)
