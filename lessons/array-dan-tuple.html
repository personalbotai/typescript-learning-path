Array di TypeScript

Array adalah kumpulan nilai yang disimpan dalam urutan tertentu. TypeScript menambahkan type annotation untuk array, sehingga kita bisa menentukan tipe elemen yang diizinkan.

1. Deklarasi Array

Beberapa cara mendeklarasikan array dengan tipe:

// Array of number
let scores: number[] = [95, 88, 76, 92];
// Atau
let scores2: Array<number> = [95, 88, 76, 92];

// Array of string
let names: string[] = ["Alice", "Bob", "Charlie"];

// Array of boolean
let flags: boolean[] = [true, false, true];

// Array bisa kosong
let empty: number[] = [];

// Mixed types (hindari jika memungkinkan)
let mixed: (string | number)[] = ["text", 123];

Hindari any[]: Jangan gunakan any[] kecuali benar-benar perlu. TypeScript kehilangan manfaat type-checking.

2. Akses dan Modifikasi Array

let fruits: string[] = ["Apple", "Banana", "Cherry"];

// Akses elemen (zero-based index)
console.log(fruits[0]); // "Apple"
console.log(fruits[2]); // "Cherry"

// Modifikasi elemen
fruits[1] = "Blueberry";
console.log(fruits); // ["Apple", "Blueberry", "Cherry"]

// Panjang array
console.log(fruits.length); // 3

// Tambah elemen di akhir
fruits.push("Durian");
console.log(fruits); // ["Apple", "Blueberry", "Cherry", "Durian"]

// Hapus elemen terakhir
let last = fruits.pop();
console.log(last); // "Durian"

// Tambah di awal
fruits.unshift("Avocado");
console.log(fruits); // ["Avocado", "Apple", ...]

// Hapus di awal
let first = fruits.shift();
console.log(first); // "Avocado"

3. Array Methods Penting

TypeScript memiliki method built-in untuk array (sama seperti JavaScript).

map()

Transform setiap elemen

let nums = [1,2,3];
let doubled = nums.map(x => x*2);
// [2,4,6]

filter()

Saring elemen berdasarkan kondisi

let nums = [1,2,3,4];
let evens = nums.filter(x => x%2===0);
// [2,4]

reduce()

Kumpulkan ke satu nilai

let nums = [1,2,3,4];
let sum = nums.reduce((a,b) => a+b, 0);
// 10

find()

Cari elemen pertama yang cocok

let nums = [1,2,3,4];
let found = nums.find(x => x>2);
// 3
let numbers: number[] = [1, 2, 3, 4, 5];

// map: kalikan setiap angka dengan 2
let doubled: number[] = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

// filter: ambil angka genap
let evens: number[] = numbers.filter(n => n % 2 === 0);
console.log(evens); // [2, 4]

// reduce: jumlahkan semua
let sum: number = numbers.reduce((acc, n) => acc + n, 0);
console.log(sum); // 15

// find: cari angka pertama > 3
let found: number | undefined = numbers.find(n => n > 3);
console.log(found); // 4

4. Tuple (Array dengan Fixed Tipe dan Length)

Tuple adalah array dengan jumlah elemen tetap dan tipe tertentu untuk setiap posisi. Berguna untuk data structur seperti koordinat atau pasangan nilai.

// Tuple: 2 elemen, pertama string, kedua number
let person: [string, number] = ["Alice", 25];
console.log(person[0]); // "Alice" (string)
console.log(person[1]); // 25 (number)

// Tuple dengan 3 elemen
let coordinate: [number, number, number] = [10, 20, 30];

// Bisa akses dengan destructuring
let [x, y, z] = coordinate;
console.log(x, y, z); // 10 20 30

// TypeScript mengecek tipe setiap posisi
person = ["Bob", "tiga puluh"]; // Error! Harus number untuk posisi 2

// Tuple bisa memiliki optional elements
let optionalTuple: [string, number?] = ["satu"]; // OK, kedua opsional

Use case: Gunakan tuple untuk data yang memiliki struktur tetap, seperti [tanggal: string, waktu: string] atau [nama: string, usia: number].

Array vs Tuple

Feature Array Tuple
Length Flexible (bisa grow/shrink) Fixed (ditentukan saat deklarasi)
Element Types Semua elemen sama (homogen) Setiap posisi bisa berbeda (heterogen)
Use Case Koleksi data serupa Data dengan struktur tetap (pair, triple)

Latihan: Manipulasi Array

Buat fungsi untuk mengubah array angka menjadi array baru dengan nilai-nilai yang sudah diproses.

function processScores(scores: number[]): { avg: number, max: number, min: number } {
    if (scores.length === 0) {
        throw new Error("Array tidak boleh kosong");
    }

    let sum: number = 0;
    let max: number = scores[0];
    let min: number = scores[0];

    for (let score of scores) {
        sum += score;
        if (score > max) max = score;
        if (score < min) min = score;
    }

    let avg: number = sum / scores.length;

    return { avg, max, min };
}

let data = [85, 90, 78, 92, 88];
let result = processScores(data);
console.log(result);
// { avg: 86.6, max: 92, min: 78 }

Kesalahan Umum

Out of bounds

let arr = [1,2,3];
console.log(arr[5]); // undefined, bukan error!

Mutasi yang tidak diinginkan

let original = [1,2,3];
let copy = original; // Ini salinan referensi, bukan deep copy!
copy.push(4);
console.log(original); // [1,2,3,4] - original ikut berubah!

Rangkuman

  • Array: kumpulan elemen homogen dengan length dinamis
  • Deklarasi: number[] atau Array<number>
  • Tuple: array dengan panjang dan tipe tetap per posisi, [string, number]
  • Gunakan array untuk koleksi data serupa, tuple untuk data berstruktur tetap
  • Pelajari method built-in: map, filter, reduce, find
  • Hati-hati dengan out-of-bounds dan mutasi tidak sengaja
