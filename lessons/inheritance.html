Inheritance (Warisan)

Inheritance memungkinkan class baru (subclass/child) mewarisi properties dan methods dari class lain (superclass/parent). Ini promotes code reuse dan establishes hierarchy.

Kata kunci: extends untuk warisan, super untuk mengakses parent class.

1. extends: Warisan Sederhana

// Parent class (superclass)
class Animal {
    constructor(public nama: string, protected umur: number) {}

    eat(): void {
        console.log(`${this.nama} sedang makan.`);
    }

    sleep(): void {
        console.log(`${this.nama} sedang tidur.`);
    }
}

// Child class (subclass) extends Animal
class Dog extends Animal {
    // Dog mewarisi properties: nama, umur
    // Dog mewarisi methods: eat(), sleep()

    // Tambah property khusus Dog
    breed: string;

    constructor(nama: string, umur: number, breed: string) {
        super(nama, umur); // panggil constructor parent
        this.breed = breed;
    }

    // Method khusus Dog
    bark(): void {
        console.log(`${this.nama} menggonggong: Woof!`);
    }

    // Override method parent
    eat(): void {
        console.log(`${this.nama} (dog) sedang makan dog food.`);
        super.eat(); // bisa panggil parent version
    }
}

let dog = new Dog("Buddy", 3, "Golden Retriever");
dog.eat();    // "Buddy (dog) sedang makan dog food." + "Buddy sedang makan."
dog.sleep(); // "Buddy sedang tidur." (warisan)
dog.bark();  // "Buddy menggonggong: Woof!"
console.log(dog.breed); // "Golden Retriever"

2. super: Mengakses Parent

super merujuk ke parent class. Digunakan untuk:

  • super() - panggil constructor parent
  • super.methodName() - panggil method parent
  • super.propertyName - akses property parent (jika accessible)
class Vehicle {
    constructor(public brand: string) {}
    start(): void {
        console.log(`${this.brand} starting...`);
    }
}

class Car extends Vehicle {
    constructor(brand: string, public doors: number) {
        super(brand); // wajib panggil super() di constructor subclass
    }

    start(): void {
        console.log("Car specific start sequence...");
        super.start(); // panggil parent method
    }
}

let car = new Car("Toyota", 4);
car.start();
// Output:
// "Car specific start sequence..."
// "Toyota starting..."

HARUS panggil super()

Jika subclass memiliki constructor, harus panggil super() sebelum mengakses this. TypeScript akan error jika lupa.

3. Override Methods

Subclass bisa mengganti (override) method dari parent dengan implementasi baru.

class Shape {
    area(): number {
        return 0;
    }
    toString(): string {
        return "Shape";
    }
}

class Circle extends Shape {
    constructor(public radius: number) {
        super();
    }

    // Override area method
    area(): number {
        return Math.PI * this.radius * this.radius;
    }

    // Override toString
    toString(): string {
        return `Circle(radius=${this.radius})`;
    }
}

let circle = new Circle(5);
console.log(circle.area());    // 78.5398... (override)
console.log(circle.toString()); // "Circle(radius=5)" (override)

Optional: Gunakan override keyword (TypeScript 4.3+) untuk explicit override. TypeScript akan error jika tidak ada method parent yang sesuai.

4. protected Members

protected members bisa diakses di dalam class dan subclass, tapi tidak dari luar.

class Person {
    constructor(public name: string, protected age: number) {}

    getInfo(): string {
        return `${this.name}, ${this.age} years old`;
    }
}

class Student extends Person {
    constructor(name: string, age: number, public studentId: string) {
        super(name, age);
    }

    // Bisa akses protected property 'age' dari parent
    isAdult(): boolean {
        return this.age >= 18;
    }

    getInfo(): string {
        return `${super.getInfo()}, ID: ${this.studentId}`;
    }
}

let student = new Student("Alice", 20, "S123");
console.log(student.isAdult()); // true (akses this.age)
console.log(student.getInfo()); // "Alice, 20 years old, ID: S123"
// console.log(student.age); // Error! protected, hanya class/subclass

5. Multilevel Inheritance

Class bisa extends class yang sudah extends class lain (chain).

class A {
    a(): void { console.log("A"); }
}
class B extends A {
    b(): void { console.log("B"); }
}
class C extends B {
    c(): void { console.log("C"); }
}

let c = new C();
c.a(); // "A" (warisan dari A ‚Üí B ‚Üí C)
c.b(); // "B"
c.c(); // "C"

TypeScript tidak support multiple inheritance (class tidak bisa extends lebih dari satu class). Gunakan interface atau composition untuk kombinasi behaviors.

Latihan: Build Vehicle Hierarchy

abstract class Vehicle {
    constructor(public brand: string, public model: string) {}

    abstract start(): void; // abstract method

    stop(): void {
        console.log(`${this.brand} ${this.model} stopped.`);
    }

    getDescription(): string {
        return `${this.brand} ${this.model}`;
    }
}

class Motorcycle extends Vehicle {
    constructor(brand: string, model: string, public hasSidecar: boolean) {
        super(brand, model);
    }

    start(): void {
        console.log(`${this.getDescription()} motorcycle starting with ${this.hasSidecar ? "sidecar" : "solo"}`);
    }

    wheelie(): void {
        console.log("Doing a wheelie!");
    }
}

class Truck extends Vehicle {
    constructor(brand: string, model: string, public cargoCapacity: number) {
        super(brand, model);
    }

    start(): void {
        console.log(`${this.getDescription()} truck starting with ${this.cargoCapacity}kg capacity`);
    }

    loadCargo(weight: number): void {
        if (weight > this.cargoCapacity) {
            console.error("Overload!");
        } else {
            console.log(`Loaded ${weight}kg`);
        }
    }
}

let moto = new Motorcycle("Honda", "CBR", false);
moto.start(); // "Honda CBR motorcycle starting with solo"
moto.wheelie();

let truck = new Truck("Isuzu", "NQR", 5000);
truck.start(); // "Isuzu NQR truck starting with 5000kg capacity"
truck.loadCargo(3000);

Rangkuman

  • Inheritance dengan class Child extends Parent
  • super() wajib di constructor subclass untuk memanggil parent constructor
  • super.method() untuk memanggil method parent (override)
  • protected members accessible di subclass
  • TypeScript tidak support multiple inheritance (single inheritance only)
  • Gunakan abstract class untuk base class yang tidak bisa di-instantiate
