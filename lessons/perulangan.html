Perulangan: for, while, do-while, for...of, for...in

Modul 1: Dasar-Dasar TypeScript

Kemajuan Belajar 0%

Apa itu Perulangan?

Perulangan (loop) memungkinkan kita mengeksekusi blok kode berulang kali tanpa menulis kode yang sama berulang. Ini esensial untuk bekerja dengan koleksi data atau melakukan tugas repetitif.

Konsep: TypeScript menggunakan loop yang sama seperti JavaScript. TypeScript menambahkan type safety pada variabel loop dan koleksi.

1. for loop ( tradisional)

Loop dengan counter yang sudah ditentukan jumlah iterasinya.

// Syntax: for (init; kondisi; increment) { ... }

// Mencetak angka 1-5
for (let i: number = 1; i <= 5; i++) {
    console.log(i);
}
// Output: 1, 2, 3, 4, 5

// Loop dengan array
let buah: string[] = ["Apel", "Jeruk", "Mangga"];
for (let index: number = 0; index < buah.length; index++) {
    console.log(buah[index]);
}
// Output: Apel, Jeruk, Mangga

Komponen for loop:

  • init: Inisialisasi variabel counter (biasanya let i = 0)
  • kondisi: Loop terus selama kondisi true
  • increment: Update counter tiap iterasi (i++, i+=2, dll)

2. while loop

Loop selama kondisi true. Cocok ketika jumlah iterasi tidak diketahui sebelumnya.

let counter: number = 0;

while (counter < 5) {
    console.log(`Counter: ${counter}`);
    counter++;
    // HATI-HATI: lupa increment â†’ infinite loop!
}

// Contoh: cari angka pertama yang habis dibagi 17
let x: number = 1;
while (x % 17 !== 0) {
    x++;
}
console.log(`Angka ${x} habis dibagi 17`); // 17

Warning: Pastikan kondisi while akan pernah menjadi false, atau akan terjadi infinite loop.

3. do-while loop

Mirip while, tapi blok dijalankan setidaknya sekali sebelum kondisi dicek.

let input: string;

do {
    input = prompt("Masukkan kata (ketik 'stop' untuk berhenti):") || "";
    console.log(`Anda mengetik: ${input}`);
} while (input !== "stop");

// Ini akan selalu mengeksekusi setidaknya 1 kali,
// bahkan jika kondisi awal sudah false.

Gunakan do-while ketika Anda butuh eksekusi minimal 1 kali, misalnya menu interaktif.

4. for...of loop (ES6)

Loop langsung melalui nilai-elemen dari iterable (array, string, Map, Set, dll). Sangat ringkas!

// Loop array
let angka: number[] = [10, 20, 30, 40];
for (let nilai of angka) {
    console.log(nilai);
}
// Output: 10, 20, 30, 40

// Loop string
for (let char of "Hello") {
    console.log(char);
}
// Output: H, e, l, l, o

// TypeScript tahu tipe elemen secara otomatis
let fruits: string[] = ["Apple", "Banana"];
for (let fruit of fruits) {
    // fruit bertipe string (inferred)
    console.log(fruit.toUpperCase());
}

Rekomendasi: Gunakan for...of untuk kebanyakan kasus loop array/string. Lebih readable dan less error-prone.

5. for...in loop

Loop melalui key/property dari object atau indeks array.

// Loop object
let orang = { nama: "Andi", umur: 25, kota: "Jakarta" };
for (let key in orang) {
    console.log(`${key}: ${orang[key]}`);
}
// Output:
// nama: Andi
// umur: 25
// kota: Jakarta

// Loop array (dapat indeks)
let colors = ["Merah", "Hijau", "Biru"];
for (let index in colors) {
    console.log(`Index ${index}: ${colors[index]}`);
}
// Output:
// Index 0: Merah
// Index 1: Hijau
// Index 2: Biru

Hati-hati: for...in akan loop semua enumerable properties, termasuk yang diwarisi dari prototype. Gunakan hasOwnProperty jika perlu filter.

Perbandingan Loop

Loop Type Kapan Digunakan Contoh
for Ketika butuh kontrol penuh (counter, step custom) for (let i=0; i
while Jumlah iterasi tidak diketahui sebelumnya while (input !== "exit")
do-while Harus dieksekusi minimal 1 kali do {...} while(condition)
for...of Loop nilai elemen (array, string, iterable) for (let x of arr)
for...in Loop keys/properties object for (let key in obj)

Latihan: Hitung Faktorial

Tulis fungsi untuk menghitung faktorial suatu angka menggunakan loop.

function faktorial(n: number): number {
    if (n < 0) {
        throw new Error("Faktorial tidak didefinisikan untuk negatif");
    }
    let hasil: number = 1;
    for (let i: number = 2; i <= n; i++) {
        hasil *= i;
    }
    return hasil;
}

console.log(faktorial(5)); // 120 (5*4*3*2*1)
console.log(faktorial(0)); // 1 (dasar matematika)

Kesalahan Umum

Off-by-one Error

// Salah: loop hanya sampai 4
for (let i = 0; i < 5; i++) { ... } // BENAR: i < 5 berarti 0,1,2,3,4 (5 iterasi)
// Atau: for (let i = 1; i <= 5; i++) { ... } // juga 5 iterasi

Infinite Loop

let i = 0;
while (i < 5) {
    console.log(i);
    // lupa i++ â†’ infinite loop!
}

Rangkuman

  • for - kontrol penuh, cocok untuk count-based loop
  • while - loop selama kondisi true, jumlah iterasi tidak tentu
  • do-while - seperti while tapi minimal 1 eksekusi
  • for...of - loop nilai elemen (array, string, iterable) - recommended untuk array
  • for...in - loop keys/properties object atau indeks array
  • Pilih loop yang tepat untuk kasus Anda, hindari infinite loop dan off-by-one errors
