Setup Environment: Node.js, VS Code, dan TypeScript Compiler

Modul 1: Dasar-Dasar TypeScript

Kemajuan Belajar 0%

Persiapan Lingkungan Pengembangan

Sebelum mulai menulis kode TypeScript, kita perlu menyiapkan beberapa tools. Tutorial ini akan memandu instalasi:

  • Node.js - Runtime JavaScript untuk menjalankan TypeScript
  • Visual Studio Code - Editor dengan dukungan TypeScript terintegrasi
  • TypeScript Compiler (tsc) - Untuk mengubah .ts menjadi .js

1. Instal Node.js

TypeScript berjalan di atas Node.js. Unduh dan instal Node.js dari situs resmi.

Pilih Versi LTS

Pilih versi LTS (Long Term Support) untuk stabilitas. Saat ini, Node.js 20.x atau 18.x direkomendasikan.

# Setelah instalasi, verifikasi instalasi:
node --version    # Misal: v20.11.0
npm --version     # Misal: 10.2.4

2. Instal Visual Studio Code

VS Code adalah editor yang sangat cocok untuk TypeScript dengan built-in support.

  1. Kunjungi code.visualstudio.com
  2. Download untuk OS Anda (Windows, macOS, Linux)
  3. Instal seperti aplikasi biasa

Tip: Instal ekstensi TypeScript ESLint dan Prettier untuk kode yang lebih rapi.

3. Instal TypeScript Compiler (tsc)

TypeScript dilengkapi dengan compiler yang bisa diinstal secara global via npm.

# Instal TypeScript secara global
npm install -g typescript

# Verifikasi instalasi
tsc --version
# Output: Version 5.x.x

Setelah instalasi, perintah tsc akan tersedia di terminal untuk mengompile file .ts.

4. Buat Project Pertama

Mari buat project TypeScript sederhana untuk memastikan semua tools berfungsi.

# 1. Buat direktori project
mkdir belajar-typescript
cd belajar-typescript

# 2. Inisialisasi package.json
npm init -y

# 3. Buat file TypeScript
echo 'let pesan: string = "Hello, TypeScript!";
console.log(pesan);' > app.ts

# 4. Compile ke JavaScript
tsc app.ts

# 5. Jalankan hasil JavaScript
node app.js

âœ… Jika berhasil, akan muncul output: Hello, TypeScript!

5. Konfigurasi TypeScript (tsconfig.json)

File tsconfig.json mengontrol perilaku compiler TypeScript.

{
  "compilerOptions": {
    "target": "ES2022",
    "module": "commonjs",
    "strict": true,
    "outDir": "./dist",
    "rootDir": "./src",
    "esModuleInterop": true
  },
  "include": ["src/**/*"]
}

Dengan konfigurasi ini:

  • src/ berisi file .ts
  • Hasil kompilasi .js masuk ke dist/
  • strict: true mengaktifkan semua pemeriksaan tipe ketat
