Functional Array Methods

TypeScript (seperti JavaScript) menyediakan banyak method built-in untuk array yang membuat manipulasi data menjadi lebih declarative dan powerful. Method-method ini biasanya mengembalikan array baru tanpa mengubah array asli (immutable).

Functional Programming: Method seperti map, filter, reduce adalah basic building blocks untuk functional style. Mereka menerima callback function dan mengembalikan hasil baru.

1. map()

Transform setiap elemen array menjadi nilai baru. Tidak mengubah array asli.

let numbers: number[] = [1, 2, 3, 4, 5];

// Kali setiap angka dengan 2
let doubled: number[] = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8, 10]
console.log(numbers); // [1, 2, 3, 4, 5] (unchanged)

// Konversi ke string
let strings: string[] = numbers.map(n => `Angka ${n}`);
console.log(strings); // ["Angka 1", "Angka 2", ...]

// Dari object ke array string
let users = [
    { nama: "Ali", umur: 20 },
    { nama: "Budi", umur: 25 }
];
let names: string[] = users.map(u => u.nama);
console.log(names); // ["Ali", "Budi"]
map<U>(callback: (value: T, index: number, array: T[]) => U): U[]

Mengembalikan array baru dengan hasil callback untuk setiap elemen.

2. filter()

Saring elemen berdasarkan kondisi. Hanya elemen yang membuat callback return true yang dimasukkan.

let numbers: number[] = [1, 2, 3, 4, 5, 6];

// Ambil angka genap
let evens: number[] = numbers.filter(n => n % 2 === 0);
console.log(evens); // [2, 4, 6]

// Ambil angka > 3
let greater: number[] = numbers.filter(n => n > 3);
console.log(greater); // [4, 5, 6]

// Dari array object
let users = [
    { nama: "Ali", umur: 17 },
    { nama: "Budi", umur: 21 },
    { nama: "Citra", umur: 19 }
];
let adults = users.filter(u => u.umur >= 21);
console.log(adults); // [{ nama: "Budi", umur: 21 }]
filter(callback: (value: T, index: number, array: T[]) => boolean): T[]

Mengembalikan array baru dengan elemen yang lolos filter.

3. reduce()

Kumpulkan semua elemen menjadi satu nilai tunggal (accumulator). Sangat fleksibel untuk summing, grouping, dll.

let numbers: number[] = [1, 2, 3, 4, 5];

// Jumlahkan semua
let sum: number = numbers.reduce((acc, n) => acc + n, 0);
console.log(sum); // 15

// Cari maksimum
let max: number = numbers.reduce((acc, n) => (n > acc ? n : acc), numbers[0]);
console.log(max); // 5

// Hitung frekuensi
let chars: string[] = ['a', 'b', 'a', 'c', 'b', 'a'];
let freq: { [key: string]: number } = chars.reduce((acc, char) => {
    acc[char] = (acc[char] || 0) + 1;
    return acc;
}, {});
console.log(freq); // { a: 3, b: 2, c: 1 }
reduce<U>(callback: (acc: U, value: T, index: number, array: T[]) => U, initial: U): U

Mengembalikan nilai akumulasi setelah iterasi semua elemen.

4. find() dan findIndex()

find() mengembalikan elemen pertama yang memenuhi kondisi. findIndex() mengembalikan indeksnya.

let numbers: number[] = [1, 2, 3, 4, 5];

// Cari angka pertama > 3
let found: number | undefined = numbers.find(n => n > 3);
console.log(found); // 4

let idx: number = numbers.findIndex(n => n > 3);
console.log(idx); // 3 (indeks angka 4)

// Jika tidak ditemukan,è¿”å›ž undefined / -1
let notFound: number | undefined = numbers.find(n => n > 10);
console.log(notFound); // undefined

let notFoundIdx: number = numbers.findIndex(n => n > 10);
console.log(notFoundIdx); // -1

5. some() dan every()

some() mengecek apakah setidaknya satu elemen memenuhi kondisi. every() mengecek apakah semua elemen memenuhi.

let numbers: number[] = [1, 2, 3, 4, 5];

// Ada yang genap?
let adaGenap: boolean = numbers.some(n => n % 2 === 0);
console.log(adaGenap); // true

// Semua positif?
let semuaPositif: boolean = numbers.every(n => n > 0);
console.log(semuaPositif); // true

// Semua > 3?
let semuaLebihDari3: boolean = numbers.every(n => n > 3);
console.log(semuaLebihDari3); // false

6. sort()

Mengurutkan array in-place (mengubah array asli). Hati-hati karena default sort adalah sebagai string!

let numbers: number[] = [3, 1, 4, 1, 5];

// SALAH: default sort sebagai string
numbers.sort();
console.log(numbers); // [1, 1, 3, 4, 5] - OK untuk angka kecil, tapi tidak selalu

// BENAR: berikan compare function
numbers.sort((a, b) => a - b); // ascending
console.log(numbers); // [1, 1, 3, 4, 5]

numbers.sort((a, b) => b - a); // descending
console.log(numbers); // [5, 4, 3, 1, 1]

// Sort string
let fruits: string[] = ["cherry", "apple", "banana"];
fruits.sort(); // alphabetical ascending
console.log(fruits); // ["apple", "banana", "cherry"]

Never forget compare function for numbers! arr.sort() tanpa callback akan convert ke string dan mengurutkan lexicographically (e.g., [10, 2] â†’ [10, 2] karena "10" < "2").

7. slice() dan splice()

slice() mengembalikan potongan array baru (immutable). splice() mengubah array asli (menghapus/tambahkan).

let arr: number[] = [1, 2, 3, 4, 5];

// slice(start, end?) - end eksklusif
let part: number[] = arr.slice(1, 4);
console.log(part); // [2, 3, 4]
console.log(arr); // [1,2,3,4,5] unchanged

// slice tanpa end: dari start sampai akhir
let tail: number[] = arr.slice(2);
console.log(tail); // [3,4,5]

// clone array dengan slice
let clone: number[] = arr.slice();

// splice(start, deleteCount, ...items) - MUTATES array
let removed: number[] = arr.splice(2, 2); // hapus 2 elemen dari index 2
console.log(removed); // [3,4]
console.log(arr); // [1,2,5] - arr berubah!

// splice juga bisa insert
arr.splice(1, 0, 99); // insert 99 di index 1
console.log(arr); // [1,99,2,5]

8. concat() dan join()

let a: number[] = [1, 2];
let b: number[] = [3, 4];

// concat: gabung array (immutable)
let combined: number[] = a.concat(b);
console.log(combined); // [1,2,3,4]
console.log(a); // [1,2] unchanged

// join: convert ke string dengan separator
let str: string = combined.join("-");
console.log(str); // "1-2-3-4"

let csv: string = combined.join(",");
console.log(csv); // "1,2,3,4"

Latihan: Pipeline Data dengan Functional Methods

Gunakan kombinasi map, filter, reduce untuk memproses data mahasiswa.

interface Mahasiswa {
    nama: string;
    nilai: number[];
}

let data: Mahasiswa[] = [
    { nama: "Ali", nilai: [80, 85, 90] },
    { nama: "Budi", nilai: [70, 75, 65] },
    { nama: "Citra", nilai: [90, 95, 92] }
];

// 1. Hitung IPK setiap mahasiswa
let ipk = data.map(m => ({
    nama: m.nama,
    ipk: m.nilai.reduce((a, b) => a + b, 0) / m.nilai.length
}));
// [{nama:"Ali", ipk:85}, {nama:"Budi", ipk:70}, {nama:"Citra", ipk:92.33}]

// 2. Filter yang IPK >= 80
let cumlaude = ipk.filter(m => m.ipk >= 80);
// [{nama:"Ali",...}, {nama:"Citra",...}]

// 3. Ekstrak hanya nama cumlaude
let namaCumlaude = cumlaude.map(m => m.nama);
console.log(namaCumlaude); // ["Ali", "Citra"]

Rangkuman

  • map() - transform elemen
  • filter() - saring elemen
  • reduce() - kumpulkan ke satu nilai
  • find() / findIndex() - cari elemen/indeks pertama
  • some() / every() - cek kondisi parsial/total
  • sort() - urutkan (berikan compare function untuk numbers!)
  • slice() - potong (immutable), splice() - hapus/tambah (mutable)
  • concat() - gabung array, join() - ke string
  • Kombinasi method-function enables powerful data pipelines
