Operator dan Ekspresi

Modul 1: Dasar-Dasar TypeScript

Kemajuan Belajar 0%

Apa itu Operator?

Operator adalah simbol khusus yang melakukan operasi pada operand (nilai atau variabel). TypeScript mendukung berbagai jenis operator seperti JavaScript.

1. Operator Aritmatika

Untuk operasi matematika dasar.

Operator Keterangan Contoh
+ Penjumlahan / konkatenasi string 5 + 3 = 8
- Pengurangan 5 - 3 = 2
* Perkalian 5 * 3 = 15
/ Pembagian 10 / 3 = 3.33
% Modulo (sisa bagi) 10 % 3 = 1
** Eksponensial (pangkat) 2 ** 3 = 8 
let a: number = 10;
let b: number = 3;

console.log(a + b);   // 13
console.log(a - b);   // 7
console.log(a * b);   // 30
console.log(a / b);   // 3.333...
console.log(a % b);   // 1
console.log(a ** 2);  // 100

2. Operator Penugasan

Untuk menetapkan nilai ke variabel, sering dikombinasikan dengan operasi lain.

= Penugasan biasa
+= Penjumlahan + penugasan
-= Pengurangan + penugasan
*= Perkalian + penugasan
/= Pembagian + penugasan
%= Modulo + penugasan 
let x: number = 10;
x += 5;   // x = 15
x *= 2;   // x = 30
x -= 3;   // x = 27

3. Operator Perbandingan

Membandingkan dua nilai, menghasilkan boolean.

Rekomendasi: Selalu gunakan === dan !== (strict equality) untuk menghindari bug karena type coercion.

4. Operator Logika

Digunakan dengan nilai boolean.

&&

AND - true jika kedua operand true

||

OR - true jika salah satu true

!

NOT - membalik nilai boolean

let a: boolean = true;
let b: boolean = false;

console.log(a && b);  // false (kedua harus true)
console.log(a || b);  // true (salah satu true)
console.log(!a);      // false (negasi)

5. Operator Lainnya

  • Operator typeof: typeof x mengembalikan tipe sebagai string
  • Operator instanceof: x instanceof Class cek apakah object adalah instance dari class
  • Operator ternary: kondisi ? nilai1 : nilai2 (akan dibahas lebih lanjut di bagian kontrol alur)
  • Operator bitwise: &, |, ~, ^, <<, >> (jarang digunakan di TypeScript tinggi)
console.log(typeof "hello");   // "string"
console.log(typeof 42);      // "number"
console.log(typeof true);    // "boolean"

Prioritas Operator (Operator Precedence)

Operator memiliki prioritas yang berbeda. Gunakan kurung () untuk mengubah urutan evaluasi.

let hasil: number;

// Prioritas: * sebelum +
hasil = 10 + 5 * 2;   // 20 (5*2=10, 10+10=20)
hasil = (10 + 5) * 2; // 30 (15*2=30)

// Logika: && sebelum ||
let logika: boolean = true || false && false; // true (&& lebih tinggi)
let logika2: boolean = (true || false) && false; // false

Latihan Interaktif

Evaluasi ekspresi berikut dan tentukan nilai serta tipe hasilnya:

1. 10 + "5" "105" (string)
2. "10" - 5 5 (number)
3. true + 1 2 (number)
4. null == undefined true (boolean)
5. null === undefined false (boolean)

Rangkuman

  • Operator aritmatika: + - * / % **
  • Operator penugasan: = += -= *= /= %=
  • Operator perbandingan: == === != !== > < >= <= (gunakan strict ===)
  • Operator logika: && || !
  • Pahami prioritas operator, gunakan kurung untuk kejelasan
  • TypeScript mengecek tipe hasil ekspresi secara otomatis
