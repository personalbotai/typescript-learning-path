Apa itu TypeScript?

TypeScript adalah superset dari JavaScript yang dikembangkan oleh Microsoft. Artinya, semua kode JavaScript valid juga merupakan kode TypeScript yang valid. TypeScript menambahkan static typing (tipe statis) ke JavaScript, yang memungkinkan pengecekan tipe pada saat compile time (waktu kompilasi), bukan hanya di runtime.

Konsep kunci: TypeScript tidak menggantikan JavaScript. Kode TypeScript dikompilasi (transpile) menjadi JavaScript yang bisa dijalankan di browser atau Node.js.

Keunggulan TypeScript

Type Safety

Mendeteksi kesalahan tipe saat kompilasi, mencegah bug yang sulit ditangkap di runtime.

Better IDE Support

Autocompletion, go-to-definition, refactoring, dan error highlighting yang lebih pintar.

Self-Documenting

Kode menjadi lebih mudah dibaca dan dipahami karena deklarasi tipe yang eksplisit.

100% JavaScript Compatible

Setiap library JavaScript bisa digunakan langsung di TypeScript.

Contoh: JavaScript vs TypeScript

Perbandingan kode yang menunjukkan bagaimana TypeScript mencegah error tipe.

JavaScript (tanpa type checking)

function tambah(a, b) {
    return a + b;
}

console.log(tambah(5, 10));      // 15 - OK
console.log(tambah("5", 10));    // "510" - Bug!

Error hanya terlihat saat runtime, bisa sulit ditrace.

TypeScript (dengan type checking)

function tambah(a: number, b: number): number {
    return a + b;
}

console.log(tambah(5, 10));      // 15 - OK
console.log(tambah("5", 10));    // Error di compile time!

Error terdeteksi sebelum kode dijalankan.

Cara Pakai TypeScript

Ada beberapa cara menggunakan TypeScript:

  1. TypeScript CLI: Install global dan compile file .ts ke .js
  2. ts-node: Jalankan file .ts langsung tanpa compile terlebih dahulu
  3. Bundler (Webpack, Vite, esbuild): Integrasi dengan build tool modern
  4. Online Playground: Coba cepat di TypeScript Playground

Latihan Interaktif

Coba ubah kode berikut dan lihat hasilnya:

// Deklarasi variabel dengan tipe eksplisit
let nama: string = "Archon";
let umur: number = 42;
let aktif: boolean = true;

// Function dengan tipe parameter dan return
function sapa(nama: string): string {
    return `Halo, ${nama}! Selamat belajar TypeScript.`;
}

console.log(sapa(nama));
