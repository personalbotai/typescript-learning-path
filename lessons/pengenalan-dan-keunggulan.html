Modul 1: Dasar-Dasar TypeScript

1. Pengenalan dan Keunggulan TypeScript

TypeScript adalah superset JavaScript yang menambahkan static typing. TypeScript membantu menulis kode yang lebih aman, mudah dibaca, dan mudah di-maintain.

Keunggulan TypeScript:

  • Type checking pada waktu kompilasi
  • Autocompletion yang lebih baik di IDE
  • Refactoring yang lebih aman
  • Self-documenting code melalui tipe
  • Kompatibel 100% dengan JavaScript

Contoh kode TypeScript:

let nama: string = "Archon";
let umur: number = 42;
let aktif: boolean = true;

function sapa(nama: string): string {
    return `Halo, ${nama}!`;
}

console.log(sapa(nama));
